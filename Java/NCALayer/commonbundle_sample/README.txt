- Код в process-ncalayer-calls.js предназначен для чтения полей и обработки данных перед отправкой в NCALayer и после возвращения ответа. Само обращение в NCALayer происходит в ncalayer.js.

- В качестве параметра callback передаются названия JS функций, которые обрабатывают полученный ответ от NCALayer. Примеры callback функций в файле process-ncalayer-calls.js

- В зависимости от типа ключа параметр keyType может иметь значение "AUTHENTICATION" для аутентификации и "SIGNATURE" для подписи.

- Если указанных методов недостаточно для вашей работы, вы можете написать собственный модуль по инструкции в директории SDK 2.0\NCALayer.
